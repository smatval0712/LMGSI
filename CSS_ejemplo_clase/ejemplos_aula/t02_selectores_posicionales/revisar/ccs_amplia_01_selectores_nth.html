Ejemplo uso nth, nth-of-type, etc..

Instrucciones

  1. Aplique el selector nth-child a la sección siguiente con id ejemplo:
    #ejemplo :nth-child(2n) { color: blue; }
  2. Inserte un titular h2 al principio de la sección ejemplo. Observe como se aplica el color de fondo a los párrafos, se ha "desplazado" el estilo, h2 también es hijo.
  3. Inserte un titular de nivel 3 antes del párrafo número 3 y otro antes del párrafo 4. Observe como se aplica el color de fondo también a las cabecera de nivel h3, hijos y pares (2n).
  4. Modifique el selector #ejemplo :nth-child(2n) a #ejemplo p:nth-child(2n). Ahora debe cunplir ser hijo par (h2 y h3 siguen siendo hijos) y además párrafo. A h3 ahora no se le aplica el estilo

Si quire aplicar el estilo a los párrafos sin tener en cuenta otro elementos que también sean hijos debemos usar el selector p:nth-of-type

  1. Modifique el selector a #ejemplo p:nth-of-type(2n). Observe que ahora se aplica el estilo a los párrafos 2n independientemente de que haya otro elememtos intercalados.
  2. Modifique el selector a #ejemplo :nth-of-type(2n). SE aplica a lso elementos pares sean del tipo que sean. Los párrafos siguen igual pero ahora se le aplica estilo a la segunda cabeceras h3.

Estos selectores se pueden aplicar a filas de tablas, o a listas. Además podemos usar otros: :first-child, :first-of-type, :last-child, :last-of-type,, etc..

Recordemos que en la expresión an+b entre paréntesis en los selectores nth, a es el tamaño del ciclo y b la ocurrencia dentro de ese ciclo. Un sólo número indica el elemento en dicha posición, y las palabras reservadas even (par) y odd (impar) son equivalentes a 2n y 2n+1.

  1. Aplique el selector ul li:nth-child(3) y observe que sólo se aplica estilo al tercer elemento de la lista en la última sección.
  2. Modifique el selector anterior a ul li:nth-child(3n) y observe que se aplique cada tres elementos.
  3. Modifique el selector anterior a ul li:nth-child(3n+1) y observe que se aplica al primer elemento de cada ciclo de tres.
