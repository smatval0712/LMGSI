Otros ejemplos de Contadores CSS

Nota: se usan varias cabeceras h1 para separar ejemplos de uso..

El uso de contadores se basa en el pseudo elemento ::before (o ::after), la propiedad content para insertar el contador y el uso de nuevas propiedades:

  • counter-reset - Creates or resets a counter
  • counter-increment - Increments a counter value
  • counter() - Adds the value of a counter to an element

Si va a numerar los apartados de nivel 2, debe iniciar el contador en una regla CSS asociada al elemento padre, e incrementarlo y mostrarlo en una propiedad del elemento a numerar usando la pseudoclase ::before

Puede numerar los apartados de nivel 2 de una sección, crear contadores anidados, numerar listas, numerar párrafos, listas de definiciones, etc. Tiene varios ejemplos en el curso, y puede obtener más información en:

  • Contadores en W3Schools
  • Ejemplos detallados

Ejemplo de numeración de un nivel

Si quiere numerar las cabeceras de nivel 2 de esta sección con class="unnivel"debe aplicar estas tres reglas CSS:

 
	section.unnivel { 
		counter-reset: nh2;
	}
	section.unnivel h2 {
		counter-increment: nh2;
	}
	section.unnivel h2::before {
		content: "Apartado " counter(nh2) ": ";
	}

Ejemplo de numeración de dos niveles

En este caso necesita dos contadores, uno para las cabeceras h2 y otro para las h3. Las cabecera de nivel 2 muestran el contador nh3, y las cabeceras h3 el contador nh2 y el nh3:

 
	/* tres reglas del contador h2 */
	.dosniveles {
	  counter-reset: nh2;  /* contador de h2 */
	}
	.dosniveles h2 {
	  counter-increment: nh2;
	}
	.dosniveles h2::before {
	  content: counter(nh2) ". ";
	  background-color: lightgreen;
	}
	
	/* las tres reglas del contador h3 */
	.dosniveles h2 {
	  counter-reset: nh3;    /* contador de h3 */
	}
	.dosniveles h3 {
	  counter-increment: nh3;
	}
	/* muestra los dos contadores */
	.dosniveles h3::before {
	  content: counter(nh2) "." counter(nh3);  
	  background-color: lightgreen;
	}

Mumeración de listas ordenadas anidadas

Los contadores son auto-anidados: si se resetea (define) un contador en un elemento descendiente se crea una nueva instancia de ese contador (un contador distinto pero con el mismo identificador).

Esto es útil por ejemplo con las listas anidadas donde es posible que no conozcamos los niveles de anidación, o estos cambien, o sean tantos que hagan el código CSS farragoso.

En este contexto es útil la función counters(): crea un string con todos los contadores que existen en ese ámbito o scope con un identificador dado separados por el string que se indica en la función: 

 
	.listas ol {
	  counter-reset: nli;
	  list-style-type: none;
	}
	.listas li::before {
	  counter-increment: nli;
	  content: counters(nli,".") ". ";
	  background-color: lightgreen;
	}
