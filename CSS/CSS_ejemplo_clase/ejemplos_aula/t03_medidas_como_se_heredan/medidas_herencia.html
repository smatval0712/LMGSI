CSS: medidas y cómo se heredan

Pequeño ejemplo aclarando cómo se heredan las medidas en CSS. En las Dev Tools del navegador se pueden consultar:

  • las reglas CSS aplicadas a un elemento.
  • los valores calculados en pixels tras aplicar esas reglas.

Observe cómo dentro de las secciones los párrafos y las cabeceras h2 y h3 tienen aplicado el mismo valor de text-indent, aunque el tamaño de fuente es distinto (y en el caso de h2 ha sido modificado en la hoja de estilos). Tiene más detalles en rl fichero CSS: medidas_herencia.css
